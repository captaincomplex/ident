# Ident

A wall-mounted flight board for Raspberry Pi. It logs in, auto-refreshes, and
renders live routes on a real coastline map with charts — all rendered on-device,
so the Pi needs no heavy geo libraries at runtime.

**Status:** In active development
**Stack:** Raspberry Pi · Python

Part of the [slop.aero](https://github.com/captaincomplex/captaincomplex.github.io) projects site.
